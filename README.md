# RecurPay

A subscription payment tracker for Nigerian small business owners built on Nomba's payment infrastructure.

## What it does
- Track customers and recurring payment due dates
- Flag overdue payments
- Generate Nomba checkout links for payment collection
- Mark payments as received

## Tech Stack
- Python 3.12
- Nomba Sandbox API (Checkout, Tokenized Cards)
- Pipedream (Webhook handling)

## Setup
1. Clone the repo
2. Create config.py with your Nomba credentials
3. Run: python recurpay.py
